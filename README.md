# React - TailWindCSS Starter

## Descrição

Este repositório oferece uma estrutura base para iniciar projetos em React com Tailwind CSS e Tailwindcss-Animated, já configurados e prontos para uso. O objetivo é otimizar o tempo de configuração, permitindo que o foco esteja diretamente no desenvolvimento da aplicação.

## Funcionalidades

- Configuração inicial do **React** já organizada.

- Integração completa com **Tailwind CSS**.

- Suporte ao **Tailwindcss-Animated** para animações simples e performáticas.

- Estrutura limpa e reutilizável, ideal para novos projetos.

- Fácil portabilidade: basta copiar e colar em uma nova pasta.
